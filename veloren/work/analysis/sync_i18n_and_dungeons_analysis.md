# 同步、國際化與地城生成深度分析 (Analysis)

## 1. 客戶端平滑同步機制 (`common/systems/src/interpolation.rs`)
為了解決網路抖動，Veloren 實作了精確的差值系統。

### 核心技術：
- **時間戳對齊**：利用伺服器與客戶端的時間差，在兩幀位置間進行線性或曲線插值。
- **預測與差值分離**：本地玩家使用客戶端預測以實現零延遲感；遠端實體使用差值以保證動作連貫。
- **高效能計算**：利用 Rayon 並行處理所有非玩家實體的狀態更新。

## 2. 動態多語言框架 (`common/i18n/src/lib.rs`)
支援高度動態化的文字呈現。

### 關鍵設計：
- **遞歸參數系統**：`LocalizationArg` 允許文字內容中嵌套另一個 `Content`，支援複雜的遊戲事件描述。
- **翻譯變體種子**：透過 `seed` 確保同一語境下的翻譯變體（Variation）在單次會話中保持一致性。

## 3. 幾何原語生成系統 (`world/src/site/gen.rs`)
地城與建築的生成並非隨機擺放方塊，而是幾何運算的結果。

### 核心機制：
- **CSG (構造實體幾何)**：透過 `Union` 與 `Without` 運算，從實心塊中「挖出」房間與走廊。
- **超二次曲面應用**：`Superquadric` 使得建築可以擁有圓角或特定的流線型邊緣，超越了傳統方塊感的限制。
- **Prefab 整合**：將程序化生成的邏輯骨架與 MagicaVoxel 製作的精細部件相結合。
