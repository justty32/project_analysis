# 流體物理、陣營系統與地城分佈深度分析 (Analysis)

## 1. 真實物理模擬：流體與飛行 (`common/src/comp/fluid_dynamics.rs`)
Veloren 的物理引擎遠超一般的方塊遊戲，具備基礎的航空動力學。

### 核心機制：
- **動態介面切換**：系統實時偵測實體是在空氣中還是液體中，並套用對應的密度參數。
- **升力模型**：滑翔翼的升力是基於攻角與翼型參數計算的。這包含對誘導拖曳力的模擬，使得玩家在轉向或俯衝時會損失能量與速度。
- **液體優先級**：定義了岩漿覆蓋水的邏輯，確保在極端地形下的物理判定一致性。

## 2. 社會互動基礎：陣營與組隊 (`common/src/comp/group.rs`)
陣營系統是遊戲內「敵我識別」與「數據同步」的核心。

### 關鍵設計：
- **角色層級**：區分了玩家、寵物與 NPC。寵物透過 `Owned` 屬性與主人綁定，並隨主人狀態同步。
- **高性能管理**：使用 `Slab` 與 `Join` 模式，在不影響主循環效能的前提下，管理數千個實體間的複雜社交關係（隊伍、仇恨陣營）。
- **事件驅動同步**：透過 `ChangeNotification` 確保網路封包只在狀態變動時發送，優化頻寬。

## 3. 地城的多樣性與世界地位 (`world/src/site/mod.rs`)
地城作為世界生成的「特殊站點」，具備獨特的元數據。

### 核心分佈：
- **群系自適應**：不同類型的地城（如 Adlet 或 Terracotta）會根據地理環境被分配到特定區域。
- **經濟影響**：雖然地城不交易，但它們的存在會影響周邊城鎮的人口遷移與物資需求（如藥水與裝備的需求量）。
