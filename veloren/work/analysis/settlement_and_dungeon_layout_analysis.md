# 城鎮與地城佈局深度分析 (Analysis)

## 1. 城鎮佈局與地塊管理 (`world/src/site/mod.rs`)
Veloren 使用基於地塊 (Plot) 的系統來構建複雜的定居點。

### 核心機制：
- **路權地塊優先**：系統會優先在生成的道路兩側 (`find_roadside_aabr`) 尋找空位，模擬現實中的城市擴張模式。
- **機率權重分派**：不同類型的建築（如酒館、農田、工作坊）有不同的生成權重，確保城鎮生態的功能多樣性。
- **城堡構造算法**：城堡並非單一模型，而是由城牆、塔樓、城門與主塔動態組合而成的。系統會先劃定一個巨大的 `Aabr`，再由內而外填充不同功能的子地塊。

## 2. 地城的多樣性實作 (`SiteKindMeta`)
地城是世界生成中具備特定「主題」的實體。

### 關鍵設計：
- **種族主題綁定**：地城的類型（如 Gnarling, Cultist, VampireCastle）直接決定了其生成器使用的體素材質與敵對 NPC 列表。
- **垂直層級規劃**：透過 Site 元數據，地城能定義其層數與深度，這為後續的 3D 原語生成提供了限制參數。
