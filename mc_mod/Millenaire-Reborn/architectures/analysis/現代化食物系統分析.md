# 現代化食物系統與狀態效果分析

## 1. 核心實作
- **食物構建器：** `src/main/java/me/devupdates/millenaireReborn/common/registry/MillFoodItemBuilder.java`
- **自定義食物類別：** `MillCustomFoodItem` (內嵌於 `MillFoodItemBuilder.java`)。
- **註冊調用：** `src/main/java/me/devupdates/millenaireReborn/common/registry/MillItems.java` (使用 `CreateItem` 方法)。

## 2. 1.21.8 組件化細節
- **消耗組件：** `net.minecraft.component.type.ConsumableComponents`。
- **食物組件：** `net.minecraft.component.type.FoodComponent`。
- **效果應用：** `net.minecraft.item.consume.ApplyEffectsConsumeEffect` (用於 Sake, Ayran 等特殊食物)。

## 3. 多劑量消耗邏輯
- **關鍵代碼：** `MillCustomFoodItem#finishUsing` (位於 `MillFoodItemBuilder.java`)。
- **歷史還原：** 透過 `AllFood` Map 定義 1.12.2 中的所有歷史食物數值。
